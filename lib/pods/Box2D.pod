
=head1 NAME

Box2D - 2D Physics Library

=head1 SYNOPSIS

 use Box2D;

=head1 DESCRIPTION

Box2D is a package of Perl modules that provide an object oriented
interface to the L<Box2D Physics Engine|http://www.box2d.org/> for
Perl 5.

=head1 OVERVIEW

Currently this module is a 1 to 1 binding.
This is still a WIP and so far here are the completed bindings:

 b2Mat22
 b2Mat33
 b2Rot
 b2Settings
 b2Transform
 b2TreeNode
 b2Vec2
 b2Vec3
 b2Version

=head1 USAGE

Have a look at the examples folder for usage examples.

The Box2D Manual and Documentation are also useful:

L<http://www.box2d.org/documentation.html>
