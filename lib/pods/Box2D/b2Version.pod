
=head1 NAME

Box2D::b2Version - Version structure.

=head1 SYNOPSIS

 my $v = Box2D::b2Version->new();
 $v->major( 1 ); # set major to 1
 $v->minor( 2 );
 $v->revision( 3 );

 # get and print version information
 printf( "%d.%d.%d\n", $v->major, $v->minor, $v->revision );

=head1 DESCRIPTION

Version structure used to obtain the version of the c++-library Box2D. See L<Box2D::b2_version|Box2D/b2_version>.

=head1 METHODS

=head2 new

 Box2D::b2Version->new();

Default constructor. Returns a C<Box2D::b2Version>

B<Note:> C<major>, C<minor> and C<revision> will return weird values if not
initialized.

=head2 major

 $major = $version->major;
 $version->major( 1 );

Returns the major part of the version triplet. It is meant to represent
significant changes when altered. Pass one numeric argument to set its value.

=head2 minor

 $minor = $version->minor;
 $version->minor( 2 );

Returns the minor part of the version triplet. It is meant to represent
incremental changes when altered. Pass one numeric argument to set its value.

=head2 revision

 $revision = $version->revision;
 $version->revision( 3 );

Returns the revision part of the version triplet. It is meant to represent
bug fixes when altered. Pass one numeric argument to set its value.
